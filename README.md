# Bezpieczna Paczka - driving rules learning game
## What is it?
"Bezpieczna paczka" is game that lets the user learn or remind driving rules that apply in Poland while delivering packages to different destinations in game.
## Contents of this repository
The project is devided between certain files:

`bin` - compiled code, e.g. *.exe file that lets you run the application,

`doc` - documentation; the *.xml documentation auto-generated by Visual Studio is held here,

`res` - resources needed for the project, like images and configuration (progress tracking) files,

`src` - source code for the application, in case of seeing and editing it.
## How to run the game?
There are two ways of running the game:
- Open from `bin/Debug/net9.0-windows/` file called `bezpieczna-paczkaApp.exe` to directly open the game, or
- Create a new solution in Visual Studio and add all of the files to the project, or
- Upload it to your favorite IDE that allows you to run C# Windows Forms application.
## How to play the game?
The rules are shown in the first level in Polish language.
